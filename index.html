Leaflet.Circle.toPolygon

Drop in

<script type="text/javascript" src="path/to/leaflet.circle.topolygon-min.js"></script>

Usage

var map = L.map(...);
var circle  = new L.Circle(coords, radius).addTo(map);
var polygon = new L.Polygon(circle.toPolygon()).addTo(map);
